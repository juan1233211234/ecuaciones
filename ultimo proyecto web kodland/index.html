las ecuaciones:

  1. las ecuaciones son una cuenta con una o mas incognitas y se resuelven buscando el valor de las mismas.

  2. dentro de las ecuaciones se utilizan las formulas que sirven para encontrar el valor de las mismas,ejemplos con la aritmetica:

    3. \[ \ (a+b)^2=a^2+b^2 \]

empezando con ecuaciones:

  1. en las ecuaciones,al poner un numero frente a una incognita,significa que el numero estara multiplicando a la misma,ejemplos:
    2. \[ \ 4*x=4x \] \[ \ x=1x \] \[ \ -x=-1x \]
  2. en casos de tener una o mas incognitas,se resuelve moviendo las mismas hacia un solo lado del igual y lo demas del otro,al hacer esto los signos se intercambian por el opuesto,ejemplos:
    3. \[ \ x+3=5 \] \[ \ x=5-3 \] \[ \ x=2 \]
  3. otro ejemplo mas dificil:
    4. \[ \ 4x+2=5x-4 \] \[ \ 4x-5x=-4-2 \] \[ \ -1x=-6 \] \[ \ x=-6/-1 \] \[ \ x=6 \]

aprendiendo a verificar ecuaciones:

  1. para verificar una ecuacion se intercambia la incognita por el valor encontrado,veamos unos ejemplos:
    2. \[ \ 3x+4-7x=8+2x-4 \] \[ \ 3x-7x-2x=8-4-4 \] \[ \ -6x=0 \] \[ \ x=\frac{0}{-6} \] \[ \ 3(0)+4-7(0)=8+2(0)-4 \] \[ \ 0+4-0=8+0-4 \] \[ \ 4=8-4 \] \[ \ 4=4 \]
  2. al darnos lo mismo de ambos lados del igual significa que esta perfectamente bien

potencia y raiz de ecuaciones:

  1. al cambiar de lado un termino cambia su signo al opuesto,eso quiere decir que al implementar potencias y raices se intercambiaran entre si cada que los movamos por el igual,ejemplos:
    2. \[ \ x^2=4 \] \[ \ x=\sqrt 4 \]
  2. ejemplo contrario:
    3. \[ \sqrt x=4 \] \[ \ x=4^2 \]
